What is JavaScript
------------------
- A dynamic, weakly-typed programming language
    - Compiled at runtime
    - Executed as part of a web page in a browser
    - Executed directly on any machine (i.e. host environment)


- Created to make web pages more dynamic
    - e.g. changing content of a page from inside a browser


- Originally named "LiveScript"
    - Later renamed JavaScript due to the popularity of Java
    
    - Totally independent from Java
        - Nothing in commong with the Java programming language
