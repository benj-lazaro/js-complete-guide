How JavaScript Is Executed
--------------------------
- Explores how JavaScript code in the browser (or in any environment) is executed


- The MOST important piece of code when executing JavaScript is the JavaScript Engine 
    - Google Chrome = V8
    - FireFox       = SpiderMonkey


- The job of the JavaScript Engine
    - Parse the JavaScript Code

    - Compile the JavaScript Code (on-the-fly) into Machine Code

    - Execute the Machine Code on a single thread
