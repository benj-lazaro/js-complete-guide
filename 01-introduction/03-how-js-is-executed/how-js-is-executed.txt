How JavaScript Is Executed
--------------------------
- The JavaScript Engine is the most important piece of code
    - It executes the JavaScript code in the browser

    - JavaScript Engine w/in popular browsers
        - V8 (Google Chrome)
        - SpiderMonkey (FireFox)

    - It's job is to
        - Parse the JavaScript code
        - Compile the code "on-they-fly" into machine code
        - Executes the machine code on a single thread
