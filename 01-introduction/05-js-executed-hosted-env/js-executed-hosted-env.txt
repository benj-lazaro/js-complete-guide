JavaScript Executes In a Hosted Environment
-------------------------------------------
- JavaScript Engine can be executed in different environments
    - Browser
    - Others (e.g. Server side)


- JavaScript on the browser side
    - Invented to create dynamic websites executed w/in the browser

    - It can 
        - Manipulate HTML documents & CSS stylesheets
        - Send HTTP requests in the background & more

    - It can NOT
        - Access a host computer's local filesystem
        - Interact with host computer's operating system, etc. 


- JavaScript on the server side
    - JavaScript Engine was extracted & turned into a stand-alone tool
        - That can run JavaScript anywhere
        - This tool is now known as "Node.js"

    - The code base is also referred as "server-side JavaScript"
        - Often used to write backend apps & web servers

    - It can
        - Execute JavaScript code on a host computer
        - Access the local filesystem
        - Interact with the operating systems

    - It can NOT
        - Manipulate HTML documents & CSS stylesheets
        
    - NOTE: A direct opposite of the "browser-side JavaScript"
