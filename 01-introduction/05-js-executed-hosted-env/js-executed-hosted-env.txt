JavaScript Executes In a Hosted Environment
-------------------------------------------
- Browser Side
    - JavaScript was inveted to create dynamic websites
        - That is executed in the browser 

    - JavaScript can manipulate
        - HTML document
        - CSS stylesheet
        - Sent background HTTP requests & more

    - JavaScript can NOT
        - Access the host computer's local filesystem
        - Interact w/ the host computer's operating system, etc.


- Server Side
    - Google's JavaScript engine V8 was extracted into a stand-alone tool
        - It enables JavaScript to be run anywhere
        - The stand-alone tool is now known as "Node.js"

    - Node.js can be
        - Executed on any machine 

        - Used to build 
            - Web back-end (server-side JavaScript) apps
            - Web servers
        
        - Used to access
            - Host computer's local filesystem
            - Host computer's operating system, etc.


    - Node.js can NOT
        - Manipulate HTML document and/or CSS stylesheet
