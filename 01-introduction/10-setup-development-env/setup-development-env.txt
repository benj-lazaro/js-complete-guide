Setting Up A Development Environment
------------------------------------
- Development environment 
    - Code editor   = VS Code
    - Browser       = Google Chrome + Browser Developer Tools


- Recommended VS Code Plugins
    - Material Icon Theme (Philipp Kief)
    - Prettier (Esben Petersen)


- VS Code User VS Workspace Settings
    - User      = applies configuration changes on all projects managed by VS Code
    - Workspace = appleis configuration changes to a specific project


- Google Chrome Settings
    - Use Incognito Mode to prevent from being distrated by other installed plugins
    
    - In the Console log
        - Uncheck "Preserve log" 
        - To ensure the console will be emptied at every time a webpage is reloaded
