Setting Up A Development Environment
------------------------------------
- Development environment
    - VS Code (code editor)
    
    - VS Code recommended plugins
        - Material Icon Theme (Philipp Kief)
        - Prettier (Esben Petersen)
    
    - Google Chrome (browser)


- VS Code User vs Workspace Settings
    - User settings
        - Implements configuration changes on ALL projects managed by VS Code

    - Workspace settings
        - Implements configuration changes on a specific project 


- Google Chrome browser settings
    - Use "Incognito Mode"
        - Prevents being distracted by other (installed) browser plugins 

    - On the "JavaScript Console" settings
        - Uncheck the checkbox "Preserve log" (if checked)

        - Ensure that the JavaScript console log is emptied
            - Every time the web page is reloaded
