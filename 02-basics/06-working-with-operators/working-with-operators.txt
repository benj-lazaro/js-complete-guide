Working With Variables & Operators
----------------------------------
- An operator has the ability
    - To manipulate a value assigned (stored) in a variable


- JavaScript mathematical operators
    - Addition          = +
    - Subtraction       = -
    - Multiplication    = *
    - Division          = /
    - Modulo            = %
    - Exponentiation    = **


- JavaScript assignment operator "="
    - Assigns (stores) a value into a variable
