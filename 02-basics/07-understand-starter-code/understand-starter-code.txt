Understanding The Starter Code
------------------------------
- A walkthrough of the attached starter code


- The code within the file "app.js"
    - Initializes the variable "currentResult"

    - Evaluates a mathematical expression
        - Stores the result to the variable "currentResult"

    - Calls the function "outputResult()"
        - Defined w/in the file "vendor.js"

        - Pass two (2) argument values
            - Value stored w/in the variable "currentResult"
            - An empty string


- The code within the function "outputResult()"
    - Receives the argument values

    - Selects two (2) HTML elements <h2> 
        - Within the HTML document "index.html"

        - Respectively, containing the HTML attribute "id" of
            - "current-result"
            - "current-calculation"

    - The content of the HTML elements <h2> are respectively overwritten with
        - The value of the variable "currentResult" from "app.js"
        - Empty string
