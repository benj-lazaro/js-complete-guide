Data Types
----------
- JavaScript's data Types
    - Numbers
        - Used for calculations and working with numbers
        - e.g. 2, -3, 22.956

    - String
        - Used for outputing results & gathering input
        - Enclosed within a pair of "", '' or `` (backticks)
        - e.g. "Hi", 'Hi', `Hi`

    - Boolean
        - Used by conditional statements
        - e.g. true, false

    - Objects
        - Use for grouped or related data
        - Helps organize data
        - e.g. { name: "Max", age: 31}

    - Arrays
        - Used of listing (unknown amount) of data
        - Elements are enclosed within a pair of [ ]
        - e.g. [1, 3, 5]
