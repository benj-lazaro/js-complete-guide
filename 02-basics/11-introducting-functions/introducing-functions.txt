Introducting Functions
----------------------
- A Function is a language construct
    - Refers to a piece of code that can be executed at a later point of time
    - Often referred as "code on demand"


- Code w/in a Function
    - It gets registered and then stored in memory
        - Waits until it is called from the main JavaScript code

    - It does NOT immediately gets executed
        - As the main JavaScript code is being read or running


- To define a Function

    Syntax:

    function <name>([parameter_name, ...]) {
        // body of code
    }


    - NOTE: The declaration of parameter variables w/in the pair of ( ) is OPTIONAL


- To return a value from a Function
    - At the last line of code w/in a Function
    
    - Use the keyword "return"
        - Followed by an expression, a variable or value 

    - NOTE: Returning a value from a Function is OPTIONAL

    Syntax:


- To call a Function from w/in the main JavaScript code
    - Use the convention:

    return [<variable_name>, <value>, <expression>];


- To call a Function from w/in the main JavaScript code
    - Use the calling convention
    
    <function_name>([<argument_value>, ...]);
