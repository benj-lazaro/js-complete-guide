An Introduction To Global And Local Scope
-----------------------------------------
- A constant, variable or data defined INSIDE a Function
    - Falls w/in the confines of Local (block) scope
    
    - It can NOT be accessed OUTSIDE of a function
        - Even an identical identifier defined outside a function


- A constant, variable or data defined OUTSIDE a Function 
    - Falls w/in the fones of Global scope 
    
    - It also can be accesssed w/in a Function


- A pure Function is one that
    - Takes some input via parameter(s)
    
    - Returns an output
        - Using the "return" statement
    
    - Does NOT directly access & manipulate anything from the Global scope
