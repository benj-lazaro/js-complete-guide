Executing Functions Indirectly
------------------------------
- Refactor the code base of "app.js"
    - Hook up an Event Listener to the constant "addBtn"
        - Defined within "vendor.js"

    - The constant "addBtn"
        - Stores the selected HTML element <button> w/ an attribute "id" of "btn-add"


- The JavaScript Method ".addEventListener()"
    - Requires two (2) arguments
        - An Event to listen to (e.g. "click")
        - Corresponding Callback Function

    - Callback Function will be executed
        - When the defined Event occurs within the selected HTML element <button>

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/EventTarget/addEventListener)


- The Function "add()"
    - Adds the value of the variable "currentResult" & constant "userInput"
        - Sum is stored to the variable "currentResult"
        - Effectively overwritting the previous value

    - The property ".value" attached to the constant "userInput"
        - Fetches the content of the selected HTML element in the constant "userInput" (i.e. <button>)

    - The Function "outputResult()"
        - Passes the variable "currentResult" & empty string as argument values

        - Renders the value to the corresponding HTML elements <h2> in "index.html"
