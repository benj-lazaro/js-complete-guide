Converting Data Types
---------------------
- The property ".value()" attached to the constant "userInput"
    - Where the HTML <input> with the attribute "id" of "input-number" is stored

    - Returns a string value
        - Despite the HTML element <input> has the attribute "type" set with the value of "number"
        - JavaScript still "reads" the content (value) as string

    - To convert a number of string data type into a real number
        - Use the JavaScript Function "parseInt()"

    - The Function "parseInt()"
        - Requires an argument value of string data type
        - Returns an integer of the specified base number (radix)
        - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/parseInt)

    Syntax:

    parseInt(<string>);
    parseInt(<string>, <radix>);


- To convert a value of number data type into a string
    - Use the Method ".toString()"

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number/toString)

    Syntax:

    <object>.toString();
    <object>.toString(<radix>);
