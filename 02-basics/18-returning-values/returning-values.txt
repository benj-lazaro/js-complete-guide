Returning Values
----------------
- The JavaScript "return" statement
    - Declared inside of a function

    - Returns a value
        - Back to the line of code that called the function

    - Then terminates the function's execution

    - (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/return)

    
    Syntax:
    return;                 // Returns nothing & terminates a function's execution
    return <expression>;    // Returns a value then terminates a function's execution
