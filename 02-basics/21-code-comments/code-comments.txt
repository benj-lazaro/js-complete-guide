Working With Code Comments
--------------------------
- JavaScript comments are implemented in two (2) ways:
  - Single-line
  - Multi-line

  Syntax:

  // This is a single-line comment
  
  /* This is
     a multi-line
     comment
  */


  - Comments w/in the JavaScript code
    - Are ignored and does NOT get executed by the JavaScript Engine


  - Content of a comment
    - Should NOT point out the obvious
    - Should contain short & to the point supplemental content


  Example:

  function getUserNumberInput() {         // Fetch & conver user-input value into an integer
    return parseInt(userInput.value);
  }

  /*  Fetch & convert user-input value
      Into an integer
  */
  function getUserNumberInput() {
    return parseInt(userInput.value);
  }
