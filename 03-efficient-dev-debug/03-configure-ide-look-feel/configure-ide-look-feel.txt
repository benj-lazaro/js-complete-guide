Configuring The IDE Look & Feel
-------------------------------
- Recommended VS Code configuration


- Color Theme
    - Preferences -> Theme -> Color Theme 
    - Dark Modern (default dark modern) = default


- Icon Theme
    - Preferences -> Theme -> File Icon Theme
    - Seti (Visual Studio Code)         = default
    - Material Icon Theme               = 3rd party extension


- Appearance
    - View -> Appearance -> Activity Bar Position
    - Hidden (preferred by the author)
