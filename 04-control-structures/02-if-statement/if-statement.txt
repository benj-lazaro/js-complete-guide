Introducing "if" Statements & Boolean (Comparison) Operators
------------------------------------------------------------
- The "if...else" statement
    - Executes the statement(s) w/in the "if" clause
        - When the conditional expression evaluates to truthy

    - Otherwise, executes the statement(s) w/in the "else" clause
        - NOTE: The "else" clause is OPTIONAL

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/if...else)


    Syntax:
    if (<condition>) {
        // if clause statements;
    } else {
        // else clause statements;
    }


- Boolean operators
    - Important for conditional (code) statements
        - Evaluates to either a boolean "true" or "false"
        
    - JavaScript conditional operators
        - Equality              ==
        - Inequality            !=
        - Strict equality       === (preferred; evaluates BOTH data type & value)
        - Strict Inequality     !==
        - Greater than          >
        - Greater than or equal >=
        - Less than             <
        - Less than oe equal    <=
        - NOT true              !
