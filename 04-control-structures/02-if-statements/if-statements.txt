if Statements & Boolean (Comparison) Operators
----------------------------------------------
- The if..else statement
  - Executes the statement(s) in the "if" clause if the condition is truthy

  - Otherwise (falsy), executeds the statement(s) in the "else" clause
    - NOTE: The "else" clause is OPTIONAL

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/if...else)


Syntax:

if (condition)
  statement1

// With an else clause
if (condition)
  statement1
else
  statement2


- Boolean Operators are IMPORTANT for Conditional Statements
  - It returns a value of "true" or "false"

    - Check for value equality                  =   ==
    - Check for value inequality                =   !=
    - Check for value AND type equality         =   === (preferred over ==)
    - Check for value AND type inequality       =   !==
    - Check for value being greater             =   >
    - Check for value being lesser              =   <
    - Check for value being greater or equal    =   >=
    - Check for value being lesser or equal     =   <=
    - Check if NOT true                         =   !
