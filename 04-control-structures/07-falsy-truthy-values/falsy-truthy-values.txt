Beyond true, false, Truthy & Falsy Values
-----------------------------------------
- JavaScript conditional statements work with:
    - Boolean true or false values
    - Truthy or Falsy values


- Falsy and Truthy is NOT a data type
    - It is how JavaScript evaluates conditional statements under the hood
    
    - JavaScript tries to coerce values to a Boolean value
        - If a Boolean value is require (i.e. Falsy or Truthy)


- Examples of a Falsy value
    - 0
    - An empty string ""
    - A value of null
    - A value of undefined
    - A value of NaN


- Examples of a Truthy value
    - Any number value including negative numbers
    - An non-empty string including the string "false"
    - All other objects or arrays (including empty ones)
