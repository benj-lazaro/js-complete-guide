Setting Up A Bigger Example Project (The Monster Killer)
--------------------------------------------------------
- To practice the "if" and related conditional statements
    - Provided a starter web app
    - Implement solution code on the empty "assets/scripts/app.js" file 


- The Monster Killer web app
    - Displays the HP (Health Points) of the:
        - Monster
        - Player (User)
            - With a "Bonus Life" button
            - A one-time use ONLY to recover HP

    - Provide functionality to the buttons for the player to:
        - Launch an "Attack"
        - Launch a "Strong Attack"
        - To "Heal"
        - To "Show Log" of past rounds
