Setting Up A Bigger Example Project (The Monster Killer)
--------------------------------------------------------
- To practice if and related conditional statements
    - Starter web app files provided
    - Implement code on the empty assets/scripts/app.js file 


- A monster killer web app
    - Displays the life or HP (Health Points) of the:
        - Monster
        - Player
            - With a Bonus Life button
            - One-time us ONLY to recover HP

    - Provide buttons for the player to: 
        - Launch an Attack
        - Launch a Strong Attack
        - To Heal
        - To Show Log of past rounds
