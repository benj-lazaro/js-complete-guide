A Bit of Theory Statements & Expressions
----------------------------------------
- The difference between an expression and a statement 
    - An expression
        - Yields a value
        - The returned value can be stored in a constant / variable
        - Often used on the right-side of an assignment operator "="
        - It is also a statement

    - A statement
        - Does NOT yield a value
        - It can NOT be used on the right-side of an assignment operator "="
