The for of Loop
---------------
- The for..of statement executes a loop that operates on 
    - A sequence of values sourced from an iterable object
        - Array
        - String
        - TypedArray
        - Map
        - Set
        - NodeList (and other DOM collections)
        - arguments object
        - generators (produced by generator functions )
        - user-defined iterables


Syntax:

    for (element_variable of iterable)
        statement


- The downside of using for..of loop
    - It does NOT provide access of an element's index
