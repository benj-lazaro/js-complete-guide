The "for" Loop
--------------
- The "for" loop creates a loop that
    - Consists of three (3) optional expressions
        - Initialization
        - Exit condition
        - Counter variable

    - Expressions enclosed w/in a pair of ( )
        - Each separated by a ";"

    - Followed by (usually a block) statement
        - To be executed w/in the loop

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/for)


    Syntax:
    for (<initialization>; <exit_condition>; <counter_variable>) {
        // statement
    }
