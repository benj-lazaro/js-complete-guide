The "for of" Loop
-----------------
- The "for..of" loop
    - Operates on a sequence of elements (values) from an iterable object
        - An iterable object includes
            - Array
            - String
            - TypedArray
            - Map
            - Set
            - NodeList (& other DOM collections)
            - Arguments object
            - Generators (produced by Gnerator Functions)
            - User-defined iterables

    - The downside of a "for..of" loop
        - Does NOT provide access to an array element's index value
        
        - A workaround is to declare a counter variable 
            - Outside of the "for..of" loop
            - Then increment the variable counter w/in the loop

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/for...of)


    Syntax:
    for (const <element> of <iterable>) {
        // statement
    }
