The "for..in" Loop
------------------
- The "for..in" loop
    - Enumerates properties (i.e. key-value pairs) of an object

    - When accessing the value of a property (i.e. key)
        - The property name MUST be of a string data type
            - (e.g. <object_name>["<property_name>"]) 

        - JavaScript accesses the value from the given property name

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/for...in)


    Syntax:
    for (const <property> in <object>) {
        // statement
    }
