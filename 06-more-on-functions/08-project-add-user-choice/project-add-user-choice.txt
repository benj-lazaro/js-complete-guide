Working On The Project: Adding User Choices To The Game
-------------------------------------------------------
- The prompt() is a window method that instructs the browser to
    - Display a dialog with an optional message
    - Prompts the user to input some text
    - Wait until the user either submits or cancels the dialog
    - Returns a string value
    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Window/prompt)


- The .toUpperCase() is a string method that
    - Returns the string value converted to uppercase
    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/toUpperCase)

    - A string value despite being a primitive value 
        - Can function as a pseudo object 
        - Thus granting the ability to access various methods
