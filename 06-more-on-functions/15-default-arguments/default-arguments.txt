Default Arguments In Functions
------------------------------
- A parameter variable in a function that received NO argument value
    - Defaults to the value of "undefined"

    - JavaScript allows for a parameter variable to be initialized w/ a default value

    - To initialize a default value to a parameter variable
        - Use the operator "="
        - Followed by the default value

    - Best practice to position the parameter variable w/ an initialized default value
        - At the last position of the parameter field

    - NOTE: ONLY applies to a parameter variable w/ an argument value of "undefined"
        - Any other argument value (including a falsy one) will be NOT be replaced w/ a default value     

    - When initializing a default value to parameter variable
        - The parameter variable(s) that came before can be used in a ternary expression that
            - Evaluates the default value that will be assigned

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Functions/Default_parameters)


    Syntax:
    function <name>(<parameter>, ..., <parameter>=<default_value>) {
        // statement
    }

    const <name> = (<parameter>, ..., <parameter>=<default_value>) => {
        // statement
    }
