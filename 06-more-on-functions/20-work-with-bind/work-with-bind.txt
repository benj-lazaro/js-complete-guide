Working With ".bind()"
----------------------
- The method ".bind()" of the "function" object that
    - Creates a new function w/ pre-configured argument value(s)
    
    - It takes at least two (2) argument values
        - 1st = A value referred by the keyword "this" (NOTE: ignore for now)
        - 2nd = 1st parameter variable to be configured; can accept an infinite number of succeeding parameters

        - NOTE: The order of the parameter variable(s)
            - Are expected to be received in the same order 
                - As the same order of the function's parameter fields it is "bind" to

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function/bind)

    Syntax:
    <function>.bind(this, "argument_1", ...);
