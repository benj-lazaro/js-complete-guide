Exploring And Changing DOM Properties
-------------------------------------
- Evaluating and manpulating Element nodes in the DOM
    - NOT all properties of Element nodes are writable


- Changes made in JavaScript is ONLY stored in memory
    - It does NOT change the elements & content of the rendered HTML document


- To discover associated methods & properties of Element nodes
    - Use console.dir()

    - Search MDN (https://developer.mozilla.org/en-US/)
        - Proceed to the "DOM Interface" section
        - Click the presented link
