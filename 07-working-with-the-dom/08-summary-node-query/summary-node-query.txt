Summary: Node Query Methods
---------------------------
- Summary of methods to reach out to DOM elements
    - NOTE: Only Elements nodes can be queried


- Special properties of the "document" object that
    - Returns specific parts of the HTML document
        - document.body             = returns the HTML element <body> & its child Element nodes
        - document.head             = returns the HTML element <head> & its child Element nodes
        - document.documentElement  = returns the HTML element <html> & its child Element nodes


- The selection method "document.querySelector()"
    - Takes any CSS selector
        - Returns the 1st matching Element node in the DOM
        - Returns "null" if NO matching Element node could be found
    
    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Document/querySelector)


- The selection method "document.getElementById()"
    - Takes the targeted HTML attribute "id" value w/out the "#" prefix
        - Since the same attribute "id" value should NOT occur more than once w/in a HTML document
        - Returns the matching Element node
        - Returns "null" if NO matching Element node could be found
        
    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Document/getElementById)


- The selection method "document.querySelectorAll()"
    - Takes any CSS selector
        - Returns ALL matching Element nodes as a static (non-live) "NodeList"
        - Returns an empty "NodeList" if NO matching Element node could be found

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Document/querySelectorAll)


- The selection method "document.getElementsByTagName()"
    - Takes a CSS Class selector
        - Returns ALL matching Element nodes as a live "HTMLCollection"
        - Returns an empty "HTMLCollection" if NO matching Element node could be found

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Document/getElementsByClassName)


- The selection method "document.getElementsByTagName()"
    - Takes a HTML tag
        - Returns ALL matching Element nodes as a live "HTMLCollection"
        - Returns an empty "HTMLCollection" if NO matching Element node could be found

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Element/getElementsByTagName)


- The selection method "document.getElementsByName()"
    - Takes the targeted assigned value of the HTML attribute "name"
        - Returns ALL matching Element nodes as a static (non-live) "NodeList"
        - Returns an empty "NodeList" if NO matching Element node could be found
        
    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/Document/getElementsByName)
