Adding Arrays To Arrays With .concat()
--------------------------------------
- It is a method of the Array instance
    - That merges two or more arrays
    - The existing arrays does NOT get modified
    - Returns a new array
    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/concat)

Syntax:

concat()
concat(value1)
concat(value1, value2)
concat(value1, value2, /* â€¦, */ valueN)
