Is It Included
--------------
- The .includes() is a method of the Array instance
    - Determines whether an array includes a certain value among its elements
    - Returns true or false as appropriate
    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/includes)

    - NOTE: Useful for checking the existence of specified primitive values in an array


Syntax:

includes(searchElement)
includes(searchElement, fromIndex)
