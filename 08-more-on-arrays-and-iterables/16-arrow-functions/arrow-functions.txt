Where Arrow Functions Shine!
----------------------------
- An arrow function used in an array method as a callback function
    - Helps make the code concise


- Use case, a single-line expression callback function
    - Drop the parenthesis for argument variable(s)

    - Drop unused argument variable(s)
        - Keep the ones ONLY used in the expression

    - Drop the pair of { }

    - Drop the return statement
        - Automatically returns the result to the calling function

    - Drop the statement's semi-colon at the end
