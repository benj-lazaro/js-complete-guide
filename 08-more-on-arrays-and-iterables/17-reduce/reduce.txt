The Important .reduce() Method
------------------------------
- The .reduce() is a method of Array instance
    - Executes a user-supplied "reducer" callback function on each element of an array in order
    - Passing in the return value from the calculation on the preceding element
    - The final result of running the reducer accross all elements is a single value

    - The array is NOT modified
    
    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/Reduce)


Syntax:

reduce(callbackFn)
reduce(callbackFn, initialValue)
