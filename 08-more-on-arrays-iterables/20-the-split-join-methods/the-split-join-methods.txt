Arrays & Strings: ".split()" & ".join()"
----------------------------------------
- The string method ".split()"
    - Chained to a string

    - Accepts a separator character as argument value

    - Splits the string into an ordered list of sub-strings
        - Each separated by the separator character

        - Optionally accepts a limit count as 2nd argument
            - Limits the number of sub-strings added into the array

    - Stores each sub-string as element of an array
    
    - Returns the array

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/split)


    Syntax:
    <string>.split();
    <string>.split(<separator>, [<limit_count>]);


- The array method ".join()"
    - Chained to an array

    - Optionally accepts a separator identifier as argument

    - Concatenates all elements of an array into a string
        - Converts ALL elements into string
        - Each element separated by the given separatpr identifier

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/join)


    Syntax:
    <array>.join();
    <array>.join(<separator>);
