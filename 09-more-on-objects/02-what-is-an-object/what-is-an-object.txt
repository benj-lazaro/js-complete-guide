What Is An Object
-----------------
- It is a core data structure in JavaScript
    - Made up of properties and methods
        - Stores data in properties
        - Actionres in methods

    - Groups related data together
        - Splots the code into logical pieces

    - Typically reflects real-world entities (e.g. DOM button, a movie)
        - Allows real-world logic to coding


- JavaScript Primitive Values
    - Numbers
    - Strings
    - Booleans
    - null
    - undefined
    - Symbol (advanced niche data type)


- JavaScript Reference Values (i.e. objects)
    - Everything else
    - { ... }
    - Arrays
    - Functions
    - DOM Nodes
    - Built-in objects
