What Is An Object
-----------------
- An Object is a core data structure in JavaScript
    - Made up of properties & methods (key-value pairs)
        - Property
            - A key assigned w/ data that is tied to the object

        - Method
            - A key assigned /w a function that is tied to the object

    - It allows to group related data together
        - Splits the code into logical pieces

    - It typically reflect real-world entities
        - Allows us to write real-world logic into code
        - e.g. a movie, a button in the DOM, etc.


- JavaScript has two (2) types of values
    - Primitive value
        - Numbers, Strings, Booleans, null, undefined, Symbol

    - Reference value (i.e. objects)
        - Everything else (that is NOT a primitive value)
        - User-defined object(s) using the Object literal notation {...}
        - Arrays, Functions, DOM nodes, built-in objects
