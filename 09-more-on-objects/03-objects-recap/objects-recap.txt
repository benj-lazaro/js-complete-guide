Objects Recap
-------------
- A recap on JavaScript objects
    - Object literal notation = { ... }

    - Object properties are key-value pairs inside an object
        - It stores static or dynamic values
        
        - It stores a function as a value on a certain key
            - This is referred to as a method

    - In VS Code, typing the name of the object followed by a "."
        - Provides a list of properties and/or methods defined in the object
