Adding, Modifying & Deleting Properties
---------------------------------------
- To add a property (on-the-fly) to an existing object
    - Use the Dot notation "."
        - Call the object name 
        - Followed by a "." (dot notation)
        - Next the property name (to be added)
        - Then by a "=" operator
        - And the assigned value

    - IMPORTANT: If the property ALREADY EXISTS
        - The previous value will overwritten (modified) by the new value

    Syntax:
    <object>.<new_property> = <assigned_value>;


- To remove an existing property from an object
    - Use the keyword "delete"
        - Next the object name
        - Followed by a "."
        - Then the property name to be removed

    Syntax:
    delete <object_name>.<property_name>;

    - NOTE: Accessing a deleted property, returns a value of "undefined"

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/delete) 


    - Assigning the value of "undefined" to the property to be removed
        - This approach technically works
            - The property still exists until it is garbage collected
            
        - NOTE: This is NOT ENCOURAGED
            - NEVER actively assign "undefined" as a value
            
            - ONLY JavaScript assigns "undefined" (by default) as a value
                - To an uninitialized variable or object property
        
        - Using the keyword "delete" is the WAY TO GO
            - When intending to delete an existing property


    - Assigning the value of "null" to the property
        - Does NOT REMOVE the property
        - It simply resets the its assigned value


- On the practice project (assets/scripts/object.js)
    - On the object "person"
        - Add a new property "isAdmin" on-the-fly
            - Assign a boolean value "true"

        - Delete the existing property "age"

    - Save the changes

    - Load / reload the browser
        - Open the browser's JavaScript console
        - The new property "isAdmin" is added to the object "person"
        - The property "age" is removed from the object "person"
