Property Types & Order
----------------------
- Besides strings, two (2) data type that can be used as a property name
    - Number
        - Positive numbers (0 or higher) are allowed
        - NO negative numbers

    - Symbol
        - To be discussed later
    
    - NOTE: Both of these data type property name will AUTOMATICALLY COERCED into String


- To access a property of Number data type
    - Use the Square bracket notation
        - Optionally enclose the number w/in a pair of " "

    - IMPORTANT: When accessing a property of Number data type
        - Using the "." notation w/o the Square bracket will NOT WORK
        - e.g. <object>.1, <objec>.357


- On the JavaScript object's order of properties
    - When viewing an object's properties using Chrome's Dev Tools
        - On Expanded mode
            - Property names are sorted alphabetically

        - On Summary mode
            - Property names are sorted by their insertion order
            - The order is based on how each property is added or defined w/in the object

    - NOTE: On an object w/c ONLY consists of Number data type properties
        - Properties are sorted according to their numerical ascending order
            - On BOTH Chrome Dev Tools' Expande & Summary mode
