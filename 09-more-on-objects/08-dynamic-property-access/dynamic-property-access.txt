Dynamic Property Access & Setting Properties Dynamically
--------------------------------------------------------
- The Square bracket notation
    - Offers a way to dynamically access the assigned value of an object's property
        - Assigning an existing object property name to a constant or variable
        - Pass the constant or variable as argument to the Square bracket notation
        
        - IMPORTANT: The constant or variable name MUST NOT be enclosed w/in a " "

    Syntax:
    const <key_name> = "<existing_property_name>"
    
    <object_name>[<key_name>];


    - Offers a way to dynamically add a new property to an object
        - Assign the new property name (to be added) in a constant or variable 
        
        - Inside the object
            - Place the Square bracket notation (property name side)
            - Pass the constant or variable name as argument
            - Then assign the value to the property

        - IMPORTANT: The constant or variable name MUST NOT be enclosed w/in a " "

    Syntax:
    const <key_name> = "<new_property_name>";

    const <object_name> = {
        <property_name>: <value>,
        [<constant_or_variable_name>]: <value>,
    };
        

- On the practice project (assets/scripts/object.js)
    - Create a new global constant named "userChosenKeyName"
        - Assign the value "level"

        - NOTE: Declaration of the constant "userChosenKeyName" 
            - MUST be positioned BEFORE the object "person" declaration

        - Inside the object "person"
            - Use the Square bracket notation (property name side)
            - Pass the constant "userChosenKeyName" as argument
            - Then assign the initial value of "null"

        - Outside the object "person"
            - Use the Square bracket notation to access the property "level"
            - Assign the value "Full-stack web developer"

    - Dynamically access the assigned value of an existing property
        - Create a new global constant named "keyName"
            - Assign the value of "name"

        - Call the object "person"
            - Followed by a Square bracket notation
            - Pass the constant "keyName" as argument 

            - IMPORTANT: The constant or variable name MUST NOT be enclosed w/in a " "
    
    - Save the changes

    - Load / reload the browser
        - Launch the browser's JavaScript console
        - View the results
