The Object Spread Operator (...)
--------------------------------
- Using the spread operator (...) in combination w/ objects
    - Extracts the properties & methods of an existing object into a new object
    
    Syntax:
    const <new_object> = { ...<source_object> };


- NOTE: If the source object has a property w/ an assigned reference value (e.g. array, object)
    - That (reference) value will NOT be a deep copy (i.e. shallow copy)

    - The assigned reference value still points to the same (reference) address in memory
        - Any changes on that reference value (source object) will be reflected on the new object's property
        
        - NOTE: This is a normal behavior of a reference value

    - To PREVENT this behavior
        - Use the spread operator (...) on the assigned reference value on a source object's property 
        - This will perform a deep-copy of that reference value to the new object's property
    
    Syntax:
    const <new_object> = { ...<source_object>, <new_object>.<property>: [...<source_object>.<property>], }; // Nested array

    const <new_object> = { ...<source_object>, <new_object>.<property>: {...<source_object>.<property>}, }; // Nested object
