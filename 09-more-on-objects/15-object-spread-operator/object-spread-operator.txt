The Object Spread Operator (...)
--------------------------------
- Using the spread operator (...) in combination w/ objects
    - Extracts the properties & methods of an existing object into a new object
    
    Syntax:
    const <new_object> = { ...<source_object> };


- NOTE: If the source object has a property w/ an assigned reference value (e.g. array, object)
    - That (reference) value will NOT be a deep copy (i.e. shallow copy)

    - The assigned reference value still points to the same (reference) address in memory
        - Any changes on that reference value (source object) will be reflected on the new object's property
        
        - NOTE: This is a normal behavior of a reference value

    - To PREVENT this behavior
        - Use the spread operator (...) on the assigned reference value on a source object's property 
        - This will perform a deep-copy of that reference value to the new object's property
    
    Syntax:
    const <new_object> = { ...<source_object>, <new_object>.<property>: [...<source_object>.<property>], }; // Nested array

    const <new_object> = { ...<source_object>, <new_object>.<property>: {...<source_object>.<property>}, }; // Nested object


- In the practice example (scripts.js)
    - Create a new object using Object literal notation
        - Declare the following propertiies
            - Property "name" assigned w/ a primitive value
            - Property "hobbies" assigned w/ a reference value (array)

        - Store the object on a global constant named "person"

    - Create a deep-copy of the object "person"
        - Using the spread operator (...)
        - Store the new object on a global constant named "person2"

    - Add a new property on-the-fly named "age" on the object "person2" 
        - Assign a primitive value

    - Push a new element in the property "hobbies" of the object "person"
        - Using the method ".push()"

        - This updates the proerty "hobbies" of both objects "person" & "person2"

    - Create a new object using Object literal notation
        - Based on a deep-copy of object "person"
            - Using the spread operator (...)
        
        - Declare a new property "age"
            - Assign w/ a primitive value

        - Perform a deep-copy of the proerty "hobbies" of object "person"
            - Using the spread operator (...)

    - Remove the last element from the property "hobbies" of object "person"
        - Using the method ".pop()"

        - Changes reflected on BOTH objects "person" & "person2"
            - BUT NOT on object "person3"

    - Save the changes

    - Load / reload the browser
        - Launch the brower's JavaScript console

        - View the results
