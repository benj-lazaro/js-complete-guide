Object Destructuring
--------------------
- It is similar to array destructuring except this time w/ objects
    - Using the spread operator (...) 
        - To pull out properties from an object
            - Store in a constant based on their existing property name

        - NOTE: The order how properties are pulled does NOT matter
            - The existing property name (key) matters

    - A REST parameter (...) can be used to collect
        - Other properties of an object that did NOT get pulled out
            - By their respective property name
            - Stored into a single name (e.g. ...otherProps)

    - To use a new (different) property name for the extracted existing property name
        - On the left-hand side { }
            - Use the ":" operator right after the existing property name
            - Followed by the new property name


    Syntax:
    const { <existing_property_name>, ..., <existing_property_name>} = <source_object>;

    const { <existing_property_name>, ...otherProps} = <source_object>;

    const { <existing_property_name>: <new_property_name> } = <source_object>;
