Object Destructuring
--------------------
- Functions similarly to array destructuring EXCEPT this time w/ objects
    - Uses the spread operator (...)
        - Pulls out a property (or properties) from an object
        - Store in a constant whose name is based on their existing property name

        - NOTE: The order how properties are pulled from the object DOES NOT MATTER
            - The name of existing property w/in an object DOES MATTER

    Syntax:
    const { <existing_property_name>, ..., <existing_property_name>} = <source_object>;


- A REST parameter (...) is used to collect other properties of an object that
    - Did NOT get pulled from the object by their respective property name
    - They will be COLLECTIVELY STORED under a single name (e.g. ...otherProps)

    Syntax:
    const { <existing_property_name>, ...otherProps} = <source_object>;


- To assign a new property name for the pulled property
    - Use the operator ":" in-between
        - The existing property name & the new property name

    Syntax:
    const { <existing_property_name>: <new_property_name> } = <source_object>;
