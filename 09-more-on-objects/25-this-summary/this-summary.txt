"this" Summary
--------------
- The keyword "this" can refer to different things depending on
    - Where it is used
    - How (if used w/in a function) a function is called


- Generally, "this" refers to the "thing" that called a function (if used inside a function)
    - Global context (e.g. global Window object)

    - An object

    - Some bound data / object
        - When the browser binds "this" to an HTML element <button> that triggered an event


- IMPORTANT: When in doubt on what "this" is currently references to:
    - Pass the keyword "this" as argument to "console.log()"

        console.log(this);


- On the example code (scripts.js)
    - Demonstrates what the keyword "this" references to based on
        - Where is it used

        - How "this" is reference when used w/in a function
            - How the function wherein the keyword "this" resides is called
