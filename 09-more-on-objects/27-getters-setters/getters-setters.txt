Getters & Setters
-----------------
- A JavaScript feature that controls how
    - To set (assign) a value to an object's property (i.e Setter)
    - To retrieve the assigned value from an object's property (i.e. Getter)

    - NOTE: BOTH or either one is NOT necessary for every single property of an object
    
    - Use case:
        - To create READ-ONLY properties
        - Add extra validtion on a property w/ a fallback value
        - Setup default transformations to a property's value


- To define a Getter w/in an object
    - Use the keyword "get"
        - Followed by a property (method) name (of your choice)
        - Next a pair of "( )"
        - Then a pair of "{ }" that contains the Getter's logic

        - NOTE: This will be accessed like a property, NOT a method
            - Do NOT use a pair of "( )" when calling the Getter

    - Inside the Getter
        - Use the keyword "return"
            - Followed by the object that called the Getter using the "this" keyword
            - Next is the internal property defined in the Setter      

    Syntax:
    <object> = {
        ...,
        get <method_name>() {
            return this._<internal_property>;
        },
        ...
    };


- To define a Setter w/in an object
    - Use the keyword "set"
        - Then the property (or method) name w/ a pair of "( )"

        - W/in the pair of "( )"
            - Define the paraemter variable(s) that will be worked on w/in the Setter

            - NOTE: This will be accessed like a property, NOT a method
                - Do NOT user a pair of "( )" when calling the Setter

        - Then a pair of "{ }" to contain the logic 

        - Inside the Setter
            - Target the object that called the Setter using the keyword "this"

            - Then a "." followed by a property name
                - NOTE: Use a "_" prefix to indicate that it is an internal property
            
            - Next is the "=" operator
                - To store the value taken from the paraemter variable        

    Syntax:
    <object> = {
        ...,
        set <method_name>(<args>) { 
            // Validation check logic w/ default value assignment here
            ...
            this._<property_name> = <arg>;
        },
        ...
    };
