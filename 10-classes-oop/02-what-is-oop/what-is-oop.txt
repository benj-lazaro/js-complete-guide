What Is Object-Oriented Programming (OOP)
-----------------------------------------
- Object-oriented programming (OOP) is a way of thinking
    - Structuring & writing code to work w/ real-life entities


- An example of a simple online shop app that works w/ the following components
    - A "ProductList" object
        - Contain methods that
            - Renders a product list
            - Fetches data from a server (database) & works w/ the "Product" object

    - A "Product" object
        - Contain details of a single product

        - Contain methods that
            - Renders a single product's details
            - Adds a product to the cart & works w/ the "ShoppingCart" object

    - A "ShoppingCart" object
        - Contain methods that
            - Renders the products added & the total amout
            - Allows a user to place on order (i.e. server communication logic)
