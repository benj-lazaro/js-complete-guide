Getting Started With OOP
------------------------
- Build a simply dummy shop
    - Build the front-end part of the shop
        - List of products

        - Click an "Add To Cart" button
            - Increments the total amount in a shopping cart

    - Implement the code using the object literal notation (for now)
        - Demonstrates the problem of using this notation
            - Showcase the need of a reusable "blueprint" in creating an object

        - Object literal notation is create for
            - Creating an object "on-the-fly"
            - Grouping some (related) data together

        - Using the notation
            - Hard to write reusable OOP code


- On the practice project (assets/scripts/app.js)
    - Create an object & store it in a global constant named "products"
        - Assign an array w/c contains objects as elements

        - Each object has the following properties:
            - "title"       = string
            - "imageUrl"    = string
            - "price"       = number
            - "description" = string

    - Create another object & store it in a global constant named "productList"
        - Move the object "products" w/in the "productList"

        - Refactor the object "products" into a property of "productList" that
            - Contains an array of objects

        - Create a method named "render"
            - NOTE: Use the method shorthand notation

        - Inside the method "render"
            - Select the HTML element <div> w/ an attribute "id" of "app"
                - Using the method "document.getElementById()"
                - Pass "app" as argument value
                - Store the returned Element node to the local constant named "renderHook"

            - Creata a new HTML element <ul>
                - Using the method "document.createElement()"
                - Pass "ul" as argument value
                - Store the returned Element node to the local constant named "prodList"

            - Add an HTML attribute "class" to the created HTML element <ul>
                - Chain the property ".className" to the constant "prodList"
                - Assign the value "product-list"

            - Iterate through the each element of this object's property "products"
                - Using a "for..of" loop
                    - Create a local constant named "prod"
                        - Stores an object that is read from the array

                - Create a new HTML element <li>
                    - Using the method "document.createElement()"
                    - Pass "li" as argument value
                    - Store the returned Element node to the local constant named "prodEl"

                - Add an HTML attribute "class" to the created element <li>
                    - Chain the property ".className" to the constant "prodEl"
                    - Assign the value "product-item"

                - Chain the method ".innerHTML" to the constant "prodEl"
                    - NOTE: Use backticks

                    - Implement an HTML element <div>
                        - Next an HTML element <img>
                            - Declare a HTML attribute "src"
                                - Assign the value of "${prod.imageUrl}

                            - Declare a HTML attribute "alt"
                                - Assign the value of "${prod.title}"

                        - Next is an HTML element <div> w/ a attribute "class" of "product-item__content"

                        - Next is an HTML element <h2>
                            - Assign the content of "${prod.title}"

                        - Next is an HTML element <h3>
                            - Assign the content of "${prod.price}"

                        - Next is an HTML element <p>
                            - Assign the content of "${prod.description}"

                        - Lastly is an HTML element <button>
                            - Assign the text content "Add to Cart"

                - Append the created element <li> to the element <ul>
                    - Chain the method ".append()" on the constant "prodList"
                    - Pass the constant "prodEl" as argument value

            - Chain the method ".append" to the constant "renderHook"
                - Pass "prodList" as argument value

    - Call the method "render()" of the object "productList"
