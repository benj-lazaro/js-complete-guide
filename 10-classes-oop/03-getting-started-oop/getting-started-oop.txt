Getting Started With OOP
------------------------
- To build a simple online dummy web app that has the following components
    - A list of products that contains the following attributes
        - A product name
        - Thes price
        - An image
        - A short description

    - An button "Add to Cart" that
        - Increments the "Total" amount for every product item clicked

    - NOTE: This web app focuses on the front-end aspect ONLY
        - A fully functional web app requires a server w/ a database


- First, implement the web app using Object literal notation
    - To showcase the problem (limitation) of using this approach
        - The need for a resuable "blueprint" for creating an object

    - The Object literal notation is best for
        - Creating an object "on-the-fly"
        - Grouping related data together

        - However, difficult to write reusable OOP code w/ it


- On the practice project (assets/scripts/app.js)
    - Use the Object literal notation to create an object
        - Store the object in the global constant named "productList"

    - Inside the object
        - Define a property named "products"
            - Assign it w/ an empty array
    
        - Inside the array
            - Create two (2) objects w/ the following properties:
                - "title"       = To be assigned w/ a String value
                - "imageUrl"    = To be assigned w/ a String value
                - "price"       = To be assigned w/ a Number value
                - "description" = To be assigned w/ a String value

            - Sample object elements are as follows:
                - "title"       = "A Pillow"
                - "imageUrl"    = "https://external-content.duckduckgo.com/iu/?u=https%3A%2F%2Ftse3.mm.bing.net%2Fth%2Fid%2FOIP.9UtdOxfFxHcVNKnkblTIBwHaHa%3Fpid%3DApi&f=1&ipt=8654694e58dc9093f53aa36c0b06801410e65dccbb6a013dbbbcb1e2c2f0d02c"
                - "price"       = 19.99
                - "description" = "A soft pillow!"

                - "title"       = "A Carpet"
                - "imageUrl"    = "https://external-content.duckduckgo.com/iu/?u=https%3A%2F%2Fwww.teahub.io%2Fphotos%2Ffull%2F200-2004153_carpet.jpg&f=1&nofb=1&ipt=32f5c6713ceed3de746e30c05ed5e48ddf8e99f1fa57abb0a6f6a42b68d91928"
                - "price"       = 89.99
                - "description" = "A carpet which you might like - or not."


        - Define a method named "render()"
            - Using the Method shorthand notation

        - Inside the method "render()"
            - Select the HTML element <div> w/ HTML attribute "id" of "app"
                - Using the method "document.getElementById()"
                - Pass "app" as argument
                - Store the returned Element node "div" in the local constant "renderHook"

            - Create an Element node "ul"
                - Using the method "document.createElement()"
                - Pass "ul" as argument
                - Store created Element node "ul" in the local constant "prodList"

            - Chain the property ".className" to the local constant "prodList"
                - Assign the value "product-list"

            - Use a "for..loop" to iterate each object element in the property "products"
                - Create the local constant "prod"
                    - To store the current object element read

                - On each iteration, create an Element node "li"
                    - Using the method "document.createElement()"
                        - Pass "li" as argument
                        - Store created Element node "li" in the local constant "prodEl"

                    - Chain the property ".className" to the local constant "prodEl"
                        - Assign the value "product-item"
        
                    - Chain the property ".innerHTML" to the local constant "prodEl"
                        - Use backticks to enclose the following HTML markup

                        - An HTML element <div>

                        - A child HTML element <img> w/
                            - HTML attribute "src" assigned w/ "${prod.imageUrl}"
                            - HTML attribute "alt" assigned w/ "${prod.title}"

                        - A child HTML element <div> w/
                            - HTML attribute "class" assigned w/ "product-item__content"

                        - A child element <h2> of the HTML element <div> w/
                            - The content "${prod.title}"

                        - A sibling child element <h2> w/
                            - The content "\$ ${prod.price}"
                            - NOTE: The "\$" is used to render a dollar sign 

                        - A sibling child element <p> w/
                            - The content "${prod.description}"

                        - A sibling child element <button> w/
                            - The content "Add to Cart"

                - Append the Element node "li" stored in the local constant "prodEl"
                    - To the element "ul" stored in the local constant "prodList"

                    - Chain the method "append()" to the local constant "prodList"
                        - Pass the local constant "prodEl" as argument

            - Append the Element node "ul" stored in the local constant "prodList"
                - To the element "div" stored in the local constant "renderHook"

                - Chain the method "append()" to the local constant "renderHook"
                    - Pass the local constant "prodList" as argument


    - Outside the Object literal notation
        - Call the method "render" of the object stored in the global constant "productList"

    - Save the changes

    - Load / reload the browser
    
    - Renders the object elements stored w/in the property "products" of the object
        - Stored in the global constant "productList" in the DOM
