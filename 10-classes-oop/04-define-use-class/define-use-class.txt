Defining & Using A First Class
------------------------------
- A Class allows to easily create an object
    - Serves as a theoretical "blueprint" of an object where
        - An object's properties & methods are defined


    - Easily create multiple & similar "looking" objects that
        - Contains different data (or values)
        - Retains identical structures

    - Serves as an alternative (NOT REPLACEMENT) for the Object literal notation

    - An object created from a Class is referred as
        - An "instance" of a Class or a Class-based object

    
- To create a Class
    - Use the keyword "class" followed by a name
        - The 1st letter of EVERY WORD MUST be in UPPERCASE
        - Whitespace characters in-between words is NOT allowed
    
    - Followed by a pair of "{ }"

    - Inside the pair of "{ }"
        - Define the Class field name (i.e. property)
            - Followed by a "=" operatpr
            - Then a default value
            - Terminate each field name w/ a ";"

        - NOTE: NOT assigning a default value to a Class field
            - JavaScript implicitly assigns an "undefined" value

    Syntax:
    class <class_name> {
        <field_name> = <default_value>;
        ...
        <field_name>;
    }


- To create an instance of (or instantiate) a Class
    - Use the keyword "new" followed by the Class name w/ a pair of "( )" 
        - The statement is terminated w/ a ";"
    
    - JavaScript IMPLICTLY returns a Class-based object 

    - Often the object is assigned to a (global or local) constant as value


    Syntax:
    new <class_name>();
    const <constant_name> = new <class_name>();


- On the practice project (assets/scripts/app.js)
    - Create a Class named "Product"


    - Inside the Class "Product"
        - Define the following Class fields
            - "title"       = default string value "DEFAULT"
            - "imageUrl"    = NO assigned default value (i.e. undefined)
            - "price"       = NO assigned default value (i.e. undefined)
            - "description" = NO assigned default value (i.e. undefined)


    - Instantiate the Class "Product" into an object
        - Pass the implicitly returned object as argument to "console.log()"

    
    - Save the chanages
        - Load / reload the browser
        - Launch the browser's JavaScript console

        - The details of the "Product" object is displayed
