Defining & Using A First Class 
------------------------------
- A Class allows to easily build an object
    - Serves as a theoretical blueprint of an object
        - Defines the object's properties & methods

    - Allows to easily create multiple & similar "looking" objects
        - Data or values of an object differs BUT retains identical structure
        

- An object created from a Class is referred as an "instance" of that Class
    - NOTE: This serves as an ALTERNATIVE to Object literal notation


- To create a Class
    - Use the keyword "class"
        - Followed by a name
            - UPPERCASE the 1st LETTER of EVERY WORD of the name
            - Whitespace in-between words is NOT allowed

        - Then a pair of "{ }" 
            - To contain the fields
            - NOTE: A Class field becomes an object property after instantiation

        - W/in the pair of "{ }"
            - Define the Class field name
            - Followed by a "=" operator
            - Then a default value
            - Terminate EACH FIELD w/ a ";"

            - NOTE: NOT assigning a default value, implicitly assigns "undefined"

    Syntax:
    class <class_name> {
        <field_name> = <default_value>;
        ...
        <field_name>;
    }


- To create an instance (i.e. object) of a Class
    - Use the keyword "new"
        - Followed by the Class name w/a pair of "( )" & a ";"
        - Implictly returns a Class-based object

    Syntax:
    new <class_name>()

    const <constant_name> = new <class_name>();

    - Create a class named "Product"
        - Based on the properies of the object (array) elements assigned

        - "price"       = NO assigned default value (i.e. undefined)
        - "description" = NO assigned default value (i.e. undefined)

    - Create an instance (i.e. object) of the Class "Product"
        - Pass the returned object as argument to "console.log()"

    - Save the changes

    - Load / reload the browser
        - Launch the browser's JavaScript console

    - An object "Product" is returned
        - Along with its properties & assigned (default) values
