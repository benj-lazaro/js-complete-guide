Working With Constructor Methods
--------------------------------
- The Constructor method of a Class
    - Creates Class-based objects that have different data but retain the same structure

    - Different data are passed as arguments to the Constructor method that are
        - Then assigned as initial values to correponding Class properties

    - IMPORTANT: A Class field inside a Constructor method is referred as a Class property

    - NOTE: Best to remove a defined Class field used w/in the Constructor method
        - As their counterpart Class property will override it

        - ONLY leave behind a Class field that is NOT INITIALIZED w/in the Constructor method


- During the instantiation of a Class 
    - JavaScript AUTOMATICALLY EXECUTES the Constructor method


- To create a Constructor method w/in a Class
    - Use the keyword "constructor"
        - Then a pair of "( )" where parameter variables are defined
        - Followed by a pair of "{ }" to hold the logic
        - NOTE: Do NOT terminate the statement w/ a ";"

    - Inside the the Constructor method's body
        - Define a Class property prefixed w/ the keyword "this."
            - Followed by the "=" operator
            - Then the parameter variable name whose value initializes the Class property
            - Terminate each statement w/ a ";"
        
        - NOTE: Keyword "this" refers to the Class of the Constructor method

    Syntax:
    class <class_name> {
        <field_name> = <default_value>;
        ...
        <field_name>;

        constructor(<parameter_variable>, ...) {
            this.<property_name> = <parameter_variable>;
            ...
            this.<property_name> = <parameter_variable>;
        }
    }


- To call the Constructor method of a Class
    - Use the keyword "new" then the Class name w/ a pair of "()"
        - Pass data (or values) as arguments
        - Terminate the statement w/ a ";"

    Syntax:
    new <class_name>(<argument_value>, ...);
