Working With Constructor Methods
--------------------------------
- The Constructor method of a Class
    - Allows to create an object from a Class w/ different values
        - BUT retain the same structure

    - It accepts values that are passed as arguments then
        - Assign these values to corresponding propeerties w/in the Constructor method

    - This prevent occurrences of 
        - Mistyping or missing out a property w/c is a common occurrence w/ Object literal notation

    - JavaScript automatically calls the Constructor method
        - When a Class is instantiated into an object


- To create a Constructor method w/in a Class
    - Use the keyword "constructor" followed by a pair of "( )"
        - Define parameter variable(s) here
        - Then a pair of "{ }" to contain the body
        - NOTE: Do NOT temrinate the statement w/ ";"

    - Inside the body of the Constructor method
        - Prefix each property w/ the keyword "this"
            - NOTE: Keyword "this" refers to the Class where the Constructor method resides

        - Followed by a "=" operator
            - Then the argument value to be assigned
        
        - Terminate each statement w/ a ";"

        - NOTE: A Class field w/in a Constructor method becomes a Class property

    Syntax:
    class <class_name> {
        <field_name> = <default_value>;
        ...
        <field_name>;

        constructor(<parameter_variable>, ...) {
            this.<property_name> = <parameter_variable>;
            ...
            this.<property_name> = <parameter_variable>;
        }
    }


- To call the Constructor method of a Class
    - Use the keyword "new" followed by the Class name
        - Then a pair of "( )"
        - Pass values as arguments
        - Terminate the statement w/ a ";"

    Syntax:
    new <class_name>(<argument_value>, ...);
