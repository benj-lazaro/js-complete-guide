Working With Constructor Methods
--------------------------------
- A constructor method is a special method of class
    - Used for creating & initializing an object instance of that class

    - It can accept arguments values
        - These values will be assigned to the corresponding fields
        - Fields that are then translated into properties after instantiating into an object

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Classes/constructor)


Syntax:

constructor() { /* … */ }
constructor(argument0) { /* … */ }
constructor(argument0, argument1) { /* … */ }
constructor(argument0, argument1, /* …, */ argumentN) { /* … */ }


- Class and constructor method help prevent mistyping and/or missing out properties
    - Which is a possibility when using Object Literal Notation
