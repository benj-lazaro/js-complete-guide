Working With Constructor Methods
--------------------------------
- A Constructor method of a Class
    - Allows to instantiate an object w/ different values

    - It accepts values as arguments
        - These values are then assigned to the correponding fields of that Class

    - Help prevent the occurrence of mistyping or missing out a Class' field (or an object's property)
        - This is a common issue when using Object literal notation


- To define a Constructor method of a Class
    - Use the keyword "constructor"
        - Followed by a pair of "( )"
        - Then a pair of "{ }"

        - IMPROTANT: DO NOT terminate the statement w/ a ";"

    - W/in the Constructor method's body
        - Prefix each Class field w/ the keyword "this."
            - Followed by a "=" operator
            - Then the corresponding parameter variable
            - End each statement w/ a ";"

    Syntax:
    class <name> {
        // Class field
        ...

        constructor(<parameter_variable>, ...) {
            this.<field_name> = <parameter_variable>;
            ...
        }
    }


- To call a Class' Constructor method
    - Use the keyword "new"
        - Followed by the Class' name
        - Then a pair of "( )"
        - Pass values as arguments
        - End the statement w/ a ";"

    Syntax:
    new <class_name>(<argument_value>, ...);
