Working With Constructor Methods
--------------------------------
- A Constructor method of a Class
    - Allows to instantiate an object w/ different values 
        - But retains the same structure

    - Accepts values as arguments
        - These values are assigned to the correponding Class fields

    - This helps prevent the occurrence of
        - A mistype or miss out a Class field w/c is a common occurrence
            - When using Object literal notation


- To create a Constructor method in a Class
    - Use the keyword "constructor"
        - Followed by a pair of "( )"
            - To define parameter variables
        
        - Then a pair of "{ }"
            - To contain the logic or body of the Constructor method

        - NOTE: Do NOT terminate the Constructor method statement w/ a ";"

    - Inside the body of the Constructor method
        - Prefix each Class field w/ the keyword "this"
            - NOTE: Keyword "this" refers to the Class of the Constructor method

        - Use the operator "=" to
            - Assign the correponding parameter variable 

        - Terminate each statement w/ a ";"

    Syntax:
    class <class_name> {
        <field_name> = <default_value>;
        ...
        <field_name>;

        constructor(<parameter_variable>, ...) {
            this.<field_name> = <parameter_variable>;
            ...
            this.<field_name> = <parameter_variable>;
        }
    }


- To call the Constructor method of a Class
    - Use the keyword "new" followed by the Class name
        - Then a pair of "( )"
        - Pass values as arguments
        - Terminate the statement w/ a ";"

    Syntax:
    new <class_name>(<argument_value>, ...);
