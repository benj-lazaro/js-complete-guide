Working With Constructor Methods
--------------------------------
- A Constructor method of a class
    - Allows to instantiate an object w/ different values

    - Accepts values as arguments
        - Then assign them to the corresponding class fields (properties)
    
    - Instantiates a new object w/ the passed values assigned
        - To the corresponding properties
    
    - Helps prevent the occurrence of mistyping or missing out property name(s)
        - A common incident when using object literal notation


- To define a constructor method w/in a class
    - Use the keyword "constructor" followed by a pair of "( )"
        - Define the parameter variable(s) w/in the pair of "( )"
        - Then a pair of "{ }" to contain the constructor method's body

        - IMPORTANT: Do NOT place a ";" at the end

    - W/in the constructor method's body
        - Prefix each class field w/ "this."
            - Followed by a "=" operator
            - Then the corresponding parameter variable
            - End the statement w/ a ";"

    Syntax:
    class <name> {
        // Fields...

        constructor(<parameter_variable>, ..., <parameter_variable>) {
            this.<field_name> = <parameter_variable>;
            ...
        }
    }


- To call a class constructor
    - Use the keyword "new"
        - Followed by the class name w/ a pair of "()"
        - Pass the corresponding values as arguments

    Syntax:
    new <class_name>(<arg1>,..., <arg_n>);
