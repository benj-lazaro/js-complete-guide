Using & Connecting Multiple Classes
-----------------------------------
- A Class can also be used to
    - Create an object w/c contains logic

    - Splits up the application logic into multiple classes w/c
        - Gets connected in some clever way

        - The order of these classes does NOT matter
            - JavaScript parses the entire code BEFORE they are executed

        - IMPORTANT: Classes MUST be DEFINED first 
            - BEFORE they're executed to instantiate an object

    - A method is defined w/in the Class definition
        - Simply declare the method name w/ a pair of "( )"
        - Followed by a pair of "{ }" w/c contains the body

    Syntax:
    class <name> = {
        // Class fields

        constructor() {
            // Constructor body
        }

        <method_name>() {
            // Method body
        }
    }


- To call the Method
    - Instantiate the Class into an object
        - Saved to a (global or loca) constant

    - Chain the method to the constant
        - Followed by a pair of "( )"
        - e.g. <constant_name>.<method_name>()

    Syntax:
    const <constant_name> = new <class_name>();

    <constant_name>.<method_name>();
