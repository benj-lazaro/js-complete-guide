Using & Connecting Multiple Classes
-----------------------------------
- A Class can also create an object w/c contains logic 
    - An application's logic can be split into 
        - Multiple Classes that gets connected in some clever way

    - NOTE: The order of these Classes DO NOT MATTER
        - JavaScript reads the entire code BEFORE execution 
    
    - IMPORTANT: A Class MUST be defined FIRST
        - BEFORE it is instantiated into an object 


- To define a method of a Class
    - Declare the method name
        - Followed by a pair of "( )"
        - Then a pair of "{ }"
        - NOTE: Do NOT terminate the statement w/ a ";"

    Syntax:
    class <name> {
        // Class field
        ...

        constructor(<parameter_variable>, ...) {
            this.<field_name> = <parameter_variable>;
            ...
        }

        <method_name>() {
            // Method body
            ...
        }
    }


- To call a Class' method
    - Instantiate a Class into an object 
        - Store the created object in a global or local constant

    - Chain the method name to the constant
        - Followed by a pair of "( )"
        - Terminate the statement w/ a ";"
    
    Example:
    const <object_name> = new <class_name>();

    <object_name>.<method_name>();
