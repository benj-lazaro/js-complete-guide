Binding Class Methods & Working With "this"
-------------------------------------------
- Implement the logic for the HTML element <button> w/ the content "Add to Cart"
    - Located w/in the method "render" of the Class "ProductItem"

    - When the button is clicked
        - Displays the message "Adding to Cart" on the browser's JavaScript console

  
- On the practice project (assets/scripts/app.js)
    - In the method "render()" of the Class "ProductItem"
    
    - Select the HTML element <button> currently stored w/in the local constant "prodEl"
        - Using the method ".querySelector()" on the constant "prodEl"
            - Pass "button" as argument
            - Store the returned Element node in a local constant "addCartButton"

        - Chain an Event listener to the local constant "addCartButton"
            - Using the method ".addEventListner()"
                - Pass the event "click"
                - Pass the callback function "this.addToCart"

            - NOTE: The keyword "this" references the Class "ProductItem"
                - The callback function / method "addToCart" will be created next

    - Outside the method "render()" of the Class "ProductItem"
        - Create a new method named "addToCart()" w/in the Class "ProductItem"
        - Pass the string value "Adding to cart" to "console.log" as argument
        - Pass the field "this.product" to "console.log" as argument

    - Switch back to the method "render()" of the Class "ProductItem"
        - Refactor the callback function "this.addToCart" by
            - Chaining the method ".bind()"
            - Passing "this" as argument

            - NOTE: The keyword "this" references the Class "ProductItem"

    - Save the changes

    - Load / reload the browser
        - Launch the JavaScript console
        
        - Clicking a product item's "Add to Cart" button
            - Displays the message "Adding to cart"
            - Display details of the clicked "ProductItem" object
