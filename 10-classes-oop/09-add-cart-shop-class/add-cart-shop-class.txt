Adding A Cart & Shop Class
--------------------------
- For this lesson, create two (2) new Classes
    - A Class named "ShoppingCart"
        - Contains a list of checked-out "ProductItem" objects
        - Calculates the sum (total amount) of selected "ProductItem" objects
        - Render a HTML element <button> w/ the content "Order Now!"

    - A Class named "Shop"
        - Renders the objects "ProductList" & "ShoppingCart" in the DOM


- On the practice project (assets/scripts/app.js)
    - Create a new Class named "ShoppingCart"

    - Inside the Class "ShoppingCart"
        - Define a Class field named "items"
        - Initialize it w/ an empty array

        - NOTE: Stores "ProductItem" objects that have been "added to cart" (i.e. checked-out)

    - Create a method named "render"

    - Inside the method "render()"
        - Create an Element node "section"
            - Using the method "document.createElement()"
            - Pass "section" as argument
        
        - Store the created Element node in the local constant "cartEl"

        - On the local constant "cartEl"
            - Chain the property ".className"
            - Assign the value "cart"

        - On the local constant "cartEl"
            - Chain the property ".innerHTML"
            
            - Assign the following HTML markup as value

            - An HTML element <h2> w/ 
                - The content "Total: \$${0}"
                - NOTE: The "${0}" is a dummy placeholder (for now) of the total amount

            - A sibling HTML element <button> w/
                - The content "Order Now!"

        - NOTE: The created Element node "section" represents the shopping cart

        - Return the Element node "section" (i.e. shopping cart)
            - Using a "return" statement


    - Create another Class named "Shop"
        - NOTE: This Class does NOT have any field

    - Create a method named "render"

    - Inside the method "render()"
        - Instantiate the Class "ShoppingCart"
            - Store the "ShoppingCart" object in the local constant "cart"

        - Call the method "render()" of the Class "ShoppingCart" that is
            - Currently stored w/in the local constant "cart"

        - Store the returned Element node "section" in the local constant "cartEl"
            - NOTE: This Element node represents the shopping cart


    - Outside the Class "Shop" at the bottom of the web app's codebase
        - Select & cut the two (2) lines of code:
        
        - Instantiates the Class "ProductList"
            - Stores the "ProductList" object to the local constant "productList"

        - Calls the method "render()" of the Class "ProductList"


    - Switch back to the method "render()" of the Class "Shop"

            - Calls the method "render()" of the Class "ProductList"

    - Switch to the method "render()" of the Class "ProductList"
        - Select & cut the line of code that
            - Declares the local constant "renderHook"


    - Switch back to the method "render()" of the Class "Shop"
        - Paste the line of code
            - Above the declaration of the local constant "cart"
