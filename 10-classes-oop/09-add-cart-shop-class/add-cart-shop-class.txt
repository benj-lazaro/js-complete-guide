Adding A Cart & Shop Class
--------------------------
- For this lesson
    - Create two (2) Classes
        - Shopping cart
            - Handles the HTML markup of the shopping cart
            - Contains the logic that calculates the price sum "ProductItem" objects

        - Shop
            - Contains the logic that renders "ProductList" & "ShoppingCart" objects in the DOM


- On the practice project (assets/scripts/app.js)
    - Create a new Class named "ShoppingCart"
       - Create a Class field named "items"
            - Initialize it w/ an empty array
            - NOTE: This holds "ProductItem" objects that have been added to the cart

        - Create a Class method named "render"
    

    - Inside the Class method "render()"
        - Create an Element node "section"
            - Using the method "document.createElement()"
            - Pass "section" as argument

        - Store the Element node "section" to the local constant "cartEl"
            - NOTE: This represents the shopping cart

        - Chain the property ".className" to the local constant "cartEl"
            - Assign the value "cart"

        - On a separate line, chain the property ".innerHTML" to local constant "cartEl"
            - Assign the following HTML markup

            - An HTML element <h2> w/ the content "Total: \$${0}"
                - NOTE: The "${0}" serve as a tenative placeholder for the total amount

            - A sibling HTML element <button> w/ the content "Order Now!"

        - Return the Element node "section" using a "return" statement


    - Create a new Class named "Shop"
        - Create a method named "render"


    - Inside the Class method "render()"
        - Instantiate the Class "ShoppingCart"
            - Store the "ShoppingCart" object in the local constant "cart"

        - Chain the Class method "render()" to the local constant "cart"
            - This calls the Class method "render()" of the Class "ShoppingCart|"

        - Store the returned Element node "section"
            - As an assigned value to the local constant "cartEl"


    - At the bottom of the web app's codebase
        - Cut two (2) lines of code
            - Instantiation of the Class "ProductList"
            - Calling its Class method "render()"

    
    - Back on the Class method "render()" of the Class "Shop"
        - At the last line of code, paste the previous lines of code

        - On the line of code that calls the Class method "render()"
            - Store the returned Element node "ul" to the local constant "prodListEl"

            - NOTE: This holds a list of "ProductItem" objects


    - On the Class method "render()" of the Class "ProductList"
        - Cut the line of code that
            - Creates the local constant "renderHook"
            - Assigns the Element node "div" as value


    - Back on the Class method "render()" of the Class "Shop"
        - On the line above the creation of the local constant "cart"

    - Back on the Class method "render()" of the Class "ProductList"

            - Chained method ".append()" of the (now gone) local constant "renderHook"
