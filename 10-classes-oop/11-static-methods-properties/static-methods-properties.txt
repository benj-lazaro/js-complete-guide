Static Methods & Properties 
---------------------------
- Static methods & properties (fields)
    - Defined w/in a Class 
        - Using the keyword "static"

    - ONLY accessible on the Class itself

    - Does NOT REQUIRE to instantiate the Class
        - In order to access the Static method or property

    - Use case:
        - Helper Classes
        - Global configuration
        - Working w/ object(s) of the same data & structure


- Instance methods & properties (fields)
    - Defined w/in a Class 
        - NOT using the keyword "static"

    - ONLY accessible on an instance (i.e. object) of a Class

    - Use case:
        - Core reusable logic
        - Working w/ object(s) of different data but have the same structure
