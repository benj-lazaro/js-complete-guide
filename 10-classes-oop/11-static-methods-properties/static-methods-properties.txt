Static Methods & Properties
---------------------------
- A Static method & property (field) of a Class
    - It is defined w/in a Class using the keyword "static"

    - It is ONLY accessible on the Class itself
        - Does NOT require an instance of a Class
        - To access a Static method or property

    - Common use case
        - A helper Class
        - For global configuration
        - Work w/ objects that have the same data & structure


- An Instance method or property of a Class
    - It is defined w/in a Class NOT using the keyword "static"

    - It is ONLY accessible on an instance of a Class
        - REQUIRE to instantiate a Class into an object
        - To access an Instance method or property

    - Common use case
        - Core reusable logic
        - Work w/ objects that have different data but the same structure
