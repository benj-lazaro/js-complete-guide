Static Methods & Properties 
---------------------------
- Static methods & properties (fields)
    - Defined w/ the keyword "static"
    
    - ONLY accessible w/in the class itself
        - NO need to instantiate the class into an object

    - Typically used on
        - Helper classes
        - Global configuration


- Instance methods & properties (fields)
    - Defined w/out the keyword "static"

    - ONLY accessible on instance (i.e. object) of a class

    - Typically used on
        - Core
        - Re-usable logic
        - Working w/ objects that holds different data but have the same structure


- On the practice project (assets/scripts/app.js)
    - Create a new class named "App"
        - Define a static field named "cart"

        - Create a new static method named "init()"
            - Instantiate the class "Shop"
                - Store the object on the local constant "shop"

            - Call the instance method ".render()" of the class "Shop"
                - Chain the instance method ".render()" to the local constant "shop"
    
    
    - Outside the class "App"
        - Call the static method "init()" of the class "App"


    - Switch to the class "Shop"
        - Change the local constant "cart" into a class field

        - Refactor references to the constant "cart" as a field (e.g. "this.cart")


    - Back to the class "App"
        - Access the static field "cart"
            - Assign the instance property ".cart" of the class "shop"

        - Create a new static method named "addProductToCart()"
            - Define a parameter variable named "product"

            - Access the static field "cart"
                - Chain the instance method ".addProduct()" of the class "ShoppingCart"
                - Pass "product" as argument value

            - NOTE: This static method serves as a proxy for
                - The instance method ".addProduct()" of the class "ShoppingCart"

    
    - Switch to the class "ProductItem"
        - On the instance method "addToCart()"
            - Remove the current body 

            - Call the static method ".addProductToCart() of the class "App"
                - Pass the field "product" of class "ProductItem" as argument 


    - Switch to the class "ShoppingCart"
        - On the instance method "addProduct()"
            - Chain the property ".innerHTML" on the field "outputTotal"

            - NOTE: This updates the HTML markup of "Total" in the DOM


    - Save the changes


    - Load / reload the browser
        - Clicking a product item's "Add to Cart" button
            - Changes the value of "Total" to 1
