Static Methods & Properties 
---------------------------
- A Static method & property (field) of a Class
    - Defined w/in a Class
        - Using the keyword "static"

    - ONLY accessible on the Class itself
        - Does NOT REQUIRE instantiating a Class
        - In order to access a Static method or property

    - Use case
        - Helper Class
        - Global configuration
        - Works w/ objects that have the same data & structure


- An Instance method or property (field) of a Class
    - Defined w/in a Class
        - NOT using the keyword "static"

    - ONLY accessible on an instance of a Class
        - REQUIRES instantiating a Class
        - In order to access a Static method or property

    - Use case
        - Core reusable logic
        - Works w/ objects that have different data but the same structure
