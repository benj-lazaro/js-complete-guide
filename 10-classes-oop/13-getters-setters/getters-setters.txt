Getters & Setters
-----------------
- For this lesson
    - Implement the logic that updates the HTML markup of "Total" w/in
        - The method "addProduct()" of the Class "ShoppingCart"
        - Using a Getter & a Setter as an elegant solution

    - A Getter that
        - Returns the total price of "ProductItem" objects placed in the cart
        - Retrieve the sum of the price of each "ProductItem" object added to the cart

    - A Setter that
        - Updates the HTML markup of "Total" w/ the current sum


- On the practice project (assets/scripts/app.js)
    - On the Class "ShoppingCart"
        - Create a Getter named "totalAmount"
        - Using the keyword "get"


    - Inside the Getter "totalAmount()"
        - Define a local constant named "sum"

        - Assign the Class field "items" as value to the local constant "sum"
            - Chain the method "reduce()" to the Instance field "items"

        - On the method "reduce()" pass the following arguments
            - 1st = An Arrow function
            - 2nd = Initial value of 0

        - On the Arrow function, define the parameter variables
            - 1st = previousValue
            - 2nd = currentItem

        - Inside the Arrow function
            - Use the "+" operator for the parameter variables
                - "previousValue"
                - "currentItem.price"

            - Return the sum stored in the local constant "sum"
                - Using a "return" statement


    - On the method "addProduct()"
        - On the HTML markup value assigned to the Class field "totalAmount"
            - Change the string literal template "${1}" w/
            - A call to the Getter "totalAmount"

        - Chain the method "toFixed()" to the Getter "totalAmount"
            - Pass 2 as arguments
            - NOTE: Ensures that the total contains two (2) decimal places

        - NOTE: Execute the Getter like accessing a Class field that is
            - Prefixed w/ the keyword "this"

        
    - Create a Setter named "cartItems"
        - Using the keyword "set"
            - Located above the Getter "totalAmount()"

        - Define a parameter variable named "value"

    
    - Inside the Setter "cartItems()"
        - Access the Class field "items" then
            - Assign the value stored w/in the parameter variable "value"

        - NOTE: This overwrites the previously stored "ProductItem" objects stored w/in
            - The Class field "items"


    - On the method "addProduct()"
        - Cut the declaration of the Class field "outputTotal"

        - Delete the remaining line of code
            - Leaving it an empty method (for now)


    - Back on the Setter "cartItems()"
        - Paste the code beneath the line that access the Class field "items"

    
    - Switch back to the method "addProdut()"
        - Create a new local constant named "updateItems"
            - Assign it w/ the value of the Class field "items"
            - Using a Spread operator (...)

        - Push a "ProductItem" object stored in the parameter variable "product"
            - To the array stored w/in the local constant "updateItems"

        - Assign the "ProductItem" object w/in the local constant "updatedItems"
            - Triggers the Setter "cartItems"

            - Updates the HTML markup of the shopping cart's "Total"
            

    - Save the chanages
        - Load / reload the browser

        - Clicking the "Add to Cart" button of a product item
            - Dynamically updates the value of "Total"
