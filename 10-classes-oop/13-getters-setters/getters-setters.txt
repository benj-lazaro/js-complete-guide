Getters & Setters
-----------------
- Next is to implement the logic that
    - Update the HTML markup w/in the method "addProduct()" of the Class "ShoppingCart"
        - Using Getters & Setters as an elegant solution

    - A Getter in the Class "ShoppingCart"
        - Retrieves the sum of the selected items placed in the cart

    - A Setter on the other hand
        - Updates the rendered value of "Total" in the DOM


- On the practice project (assets/scripts/app.js)
    - On the Class "ShoppingCart"
        - Create a Getter named "totalAmount"

        - In the Getter "totalAmount()"
            - Create a local constant named "sum"

            - Assign the Instance field "items"
                - Prefixed w/ the keyword "this"

            - Chain the method ".reduce()"
                - Pass the arguments
                    - 1st = an arrow function
                    - 2nd = initial value 0

            - On the arrow function's parameter field
                - Define the parameter variables
                    - 1st = previousValue
                    - 2nd = currentItem

            - Inside the arrow function
                - Return the sum of "previousValue" & "currentItem.price"
                    - Using a "return" statement

                - NOTE: The "currentItem" holds the "ProductItem" object stored w/in
                    - The Instance field "items" that has a property ".price"

            - Return the sum stored in the local constant "sum"
                - Using a "return" statement


        - On the Instance method "addProduct()"
            - On the property ".innerHTML" of the Instance field "outputTotal"
                - Replace the value 1 w/ a call to the Getter "totalAmount"
                    - Prefixed w/ the keyword "this"

                - NOTE: Treat a Getter like a property when calling it


        - Create a Setter named "cartItems"
            - Define a parameter variable named "value"
            
        - In the Setter "cartItems()"
            - Access the Instance field "items"
                - Assign w/ the value of the parameter "value"
            
            - NOTE: Overwrites previous "ProductItem" objects w/in the Instance field "items"


        - Switch back to the Instance method "addProduct()"
            - Cut the Instance field "outputTotal"

        - Switch to the Setter "cartItems()"
            - Paste the code
                - Right below the Instance field "items"

            - Chain the method ".toFixed()"
                - On the call to the Getter "totalAmount()"
                - Pass 2 as argument

        
        - Switch back to the Instance method "addProduct()"
            - Replace the remaining line of code w/ 
                - Creating a new local constant named "updateItems"
                
                - Assign a copy of the Instance field "items" contents as value
                    - Using a spread operator (...)

            - Access the Instance field "updatedItems"
                - Chain the method ".push()"
                - Pass the parameter variable "product" as argument

            - Then assign the Instance field "updatedItems" as value
                - To the Setter "cartItems"

                - NOTE: This triggers the Setter "cartItems()"

    - Save the changes

    - Load / reload the browser
        - Clicking the "Add to Cart" button of a product
        - Updates the value of "Total"
