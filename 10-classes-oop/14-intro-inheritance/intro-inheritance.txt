Introducing Inheritance
-----------------------
- The concept of Inheritance
    - To conviniently share code amongst Classes

    - Create a Base Class that
        - Contains properties & methods that are needed by other Classes

    - Extend the Base Class into a specialized Sub-Class(es)
    
    - A Sub-Class inherits the properties & methods of the Base Class
        - In addition, it can have its own specific properties & methods
        - NOTE: It can also OVERRIDE the inherited properties & methods from the Base Class
    
    - Inheritance is better than building multiple Classes that
        - Contains code duplication (i.e. properties & methods) w/c is sub-optimal
