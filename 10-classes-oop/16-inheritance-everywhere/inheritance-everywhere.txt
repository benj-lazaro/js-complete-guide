Using Inheritance Everywhere
----------------------------
- For this lesson
    - Implement the concept of inheritance on the Class "ProductItem" & "ProductList"


- On the practice project (assets/scripts/app.js)
    - On the Class "ProductItem"
        - Convert it to a Sub-Class of the Base Class "Component"

    - On the Class method "render()"
        - Delete two (2) lines of code
            - Declaration of the local constant "prodEl"

            - Chaining of the property ".className" to the local constant "prodEl"
                - Assigning of the value "product-item"

            - Leave the line of code where property ".innerHTML"
                - Chained to the loacl constant "prodEl" behind

        - Replace those lines w/ a call 
            - To the Base Class "Component" method "createRootElement()"
            
            - Pass the following arguments
                - 1st = "li"
                - 2nd = "product-item"

            - Store the returned Element node "li" 
                - As value to the (new) local constant "prodEl"

        - Delete the line of code that 
            - Returns the local constant "prodEl" using a "return" statement


    - On the Class "ProductList" method "render()"
        - Inside the method's "for..of" loop
            - Delete the line of code that chains the method "append()"
                - To the local constant "prodList"

            - Delete the keyword "const" & constant "prodEl"
                - Leaving the call to the Class "ProductItem" method "render()" behind

        - Outside the method's "for..of" loop
            - Chain a new property named "id" to the local constant "prodList"
                - Assign it w/ the value of "prod-list"

                - Located right after the chaining of the property ".className"
                    - To the local constant "prodList"

        - Back inside the method's "for..of" loop
            - On the local constant "productItem" where it is assigned w/
                - The instantiation of the Sub-Class "ProductItem"
                - Pass the value "prod-list" as 2nd argument  


    - On the (now) Sub-Class "ProductItem" Constructor method
        - Define a 2nd parameter variable named "renderHookId"

        - On the 1st line of code
            - Use the keyword "super()"
            - Pass the parameter variable "renderHookId" as argument

        - NOTE: Remember Base Class properties is REQUIRED to come FIRST 
            - BEFORE any of the Sub-Class propertie


    - On the Class "ProductList"
        - Convert it to a Sub-Class of the Base Class "Component"

        - On the Class method "render()"
            - Refactor the assignment of the local constant "prodList" to calling
                - The Base Class "Component" method "createRootElement()"

                - Pass the arguments
                    - 1st = "ul"
                    - 2nd = "product-list"
                    - 3rd = "[new ElementAttribute("id", "prod-list")]"

                - Delete the keyword "const" & constant "prodList"
                
            - Delete the lines of code that chains 
                - The property ".id" to the local constant "prodList"
                - The property ".className" to the local constant "prodList"

            - Delete the line of code that 
                - Returns the local constant "prodList" using a "return" statement

        - On the Constructor method
            - Define the parameter variable named "renderHookId"

        - Inside the Constructor method
            - Use the keyword "super()"
            - Pass the parameter variable "renderHookId" as argument


    - On the Class "Shop" method "render()"
        - Remove the declaration of the local constant "prodListEl"
            - Leaving behind the call to the Sub-Class "ProductList" method "render()"

        - Delete the line of code that
            - Chain the method "append()" to the local constant "renderHook"

        - Delete the declaration of the local constant "renderHook"

        - On the instantiation of the Sub-Class "ProductList"
            - Pass "app" as argument

            - NOTE: The value of "app" refers to the HTML element attribute "id"
                - Where the Element node "ul" will be rendered


    - Save the changes

    - Load / reload the browser
        - No changes in the behavior of the app
