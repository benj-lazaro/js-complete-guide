Builtin Classes
---------------
- The built-in global Class "Object" that creates an empty object
    - Equivalent to using Object literal notation
        - To create an empty object

            const <name> = {};

    - However, Object literal notation is preferred
        - As the global Class "Object" is a little slower in terms of performance

    - Reference (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object)


    Syntax:
    const <name> = new Object();
