First Project Steps & Planning
------------------------------
- First steps includes
    - Taking the OOP approach to plan the app's design

    - Define a Class that
        - List & manage ALL projects (Active Projects & Finished Projects)

        - Represent a single project item

        - Manage the app overall

        - Display "Tool Tips"
            - When a project item's button "More Info" is clicked

    - The app alreasy comes w/ a HTML document that
        - Contains content

        - Depicts content retrieved from a remote (backend) server

        - Simulates a real-life use case
            - Working w/ generated content
            - Using JavaScript to implement interactions w/ the content
    
    - HTML document's data of interest
        - HTML element <section> attribute "id"

    - Implement logic that
        - Parses the HTML document
        - Select HTML element <section>
        - Identify w/c project items belongs to "Active Projects" & "Finished Projects"


- On the practice project (assets/scripts/app.js)
    - Define Classes using the OOP approach
        - Class "Tooltip"
            - Handles the button "More Info" or a "ProjectItem" object

        - Class "ProjectItem"
            - Represents a project item

        - Class "ProjectList"
            - Manages "ProjectItem" objects

        - Class "App"
            - Manage the app 

    - Save the changes
