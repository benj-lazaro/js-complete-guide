Passing Method References Around
--------------------------------
- For this lesson
    - Implement the logic of the Class "ProjectList" method "addProject()"
    
    - Call the method "addProject()" 
        - From another instance of the Class "ProjectList"
        - Using a callback function
        

- On the practice project (assets/scripts/app.js)
    - On the Class "ProjectList" Constructor method
        - Define a new parameter variable "type"
        
        - Define a new Class property "this.type"
            - Assign as value the parameter variable "type"


    - On the Class "ProjectList"
        - Define a new Class method "setSwitchHandlerFunction"
            - Define a parameter variable "switchHandlerFunction" that
            - Serves as a pointer to a callback function


    - On the Class "ProjectList" method "setSwitchHandlerFunction()"
        - Define a parameter variable "switchHandlerFunction"

        - Define a new Class property "this.switchHandler"
            - Assign the parameter variable "switchHandlerFunction" as value


    - On the Class "ProjectList" method "switchProject()"
        - Access the Class property "this.switchHandler"
            - Treat it like a method by adding a pair of "( )"

        - Access the Class field "this.projects"
            - Chain the method ".find()"
                - Pass an Anonymous function as argument value

            - On Anonymous function 
                - Define the parameter variable "p"

                - Check if the property "id" of "p" ("ProjectItem" object ) is 
                    - STRICTLY EQUAL to that of the parameter variable "projectId"


    - On the Class "ProjectItem" Constructor method
        - Define a new parameter variable "updateProjectListsFunction"

        - Define a new Class property "updateProjectListsHandler"
            - Assign the parameter variable "updateProjectListsFunction" as value


    - On the Class "ProjectList" Constructor method
        - On the instantiation of the Class "ProjectItem"
            - Pass the Class method "switchProject" as the 2nd argument value
            
            - Chain the method ".bind()"
                - Pass the keyword "this" as argument value


    - On the Class "ProjectItem" method "connectSwitchButton()"
        - Pass the class property "updateProjectListsHandler" as
            - The callback function for the Event listener of the local constant "switchBtn"


    - Save the changes
        - Load / reload the browser

        - Clicking the button "Finish" of a "ProjectItem" object
            - Under the group "Active Projects"

            - Displays a "ProjectList" object w/ 
                - The value of the property "type" set to "finished"

        - Clicking the button "Activate" of a "ProjectItem" object
            - Under the group "Finished Projects"
            
            - Displays a "ProjectList" object w/ 
                - The value of the property "type" set to "active"
