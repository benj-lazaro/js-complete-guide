Using "dataset" (data-* Attributes)
-----------------------------------
- The HTML attribute "data-<name>" is a special attribute that
    - Allows any kind of data to be attached to an HTML (or a DOM) element

    - To read data from a HTML attribute "data-<name>"
        - JavaScript offers a special property named ".dataset"

    - ALL HTML attributes "data-<name>" attached to an HTML element are
        - Merged together & stored w/in the property ".dataset"

    - IMPORTANT: JavaScript automatically 
        - Converts the attribute name into camelCase
        - Removes the "data-" part of the attribute
        - (e.g. "data-extra-info" converted to "extraInfo")


- To set a new HTML attribute "data-<name>" to an HTML element
    - Select the target HTML element
        - Chain the property ".dataset"
        - Followed by a new "data-<name>"
        - Then the "=" operator
        - Last, would be the value
        - (e.g. <selected_element>.dataset.<new_name> = "<data>";)
