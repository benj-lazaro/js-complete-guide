Work With Element Sizes & Positions
-----------------------------------
- Properites that allow to work w/ specific data of the selected DOM element
    - ".offsetTop" & ".offsetLeft"
        - Returns the distance (in pixels)
            - Between the element's top-most & left-most points
            - To the start of the two-dimensional coordinate system (0,0)
            - NOT the start of the viewport

            - NOTE: Provides the "outer" positioning

    - ".clientTop" & ".clientLeft"
        - Returns the distance (in pixels)
            - Between the element's top-most & left-most points
            - To the top-most & left-most points of the element box's content 

            - NOTE: Provides the "inner positioning"
                - Content refers to the entire content w/out any border & potential scroll bars

    - ".offsetWidth" & ".offsetHeight"
        - Returns the element box's entire width & height
        - Including ALL borders & scroll bars

    - ".clientWidth" & ".clientHeight"
        - Returns the element box's inner width & height w/out
        - The borders & scroll bars

    - ".scrollHeight"
        - Returns the content's entire height w/c
        - Includes the part that is (currently) NOT visible in the box

    - ".scrollTop"
        - Returns information on how much content in the box that
            - Have been scrolled down

        - NOTE: The value of 0 meant that the content
            - Has NOT (yet) been scrolled down

    - Other related properties can be viewed from the following references
        - HTMLElement Reference (https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement)
        - Element Reference (https://developer.mozilla.org/en-US/docs/Web/API/Element)


- Properties that retrieves the width & height of an HTML document
    - "window.innerWidth" & "window.innerHeight"
        - Returns the HTML document's width & height w/in
            - The "window" object
            - Excludes the browser's URL bar

        - NOTE: If the browser has a visible scroll bar (i.e. IE browser or Windows)
            - The returned value INCLUDES the size of the scroll bar
            - Instead of being subtracted from the returned width & height values

    - "document.documentElement.clientWidth" & "document.documentElement.clientHeight"
        - Returns the HTML document's "REAL" width & height
        
        - NOTE: If the browser has a visible scroll bar
            - The returned value EXCLUDES the size of the scroll bar
            - Implicitly subtracts from the returned width & height values
           
    - NOTE: The "documentElement" refers to the root element w/c includes the head & body
