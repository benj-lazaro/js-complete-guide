The DOM & Prototypes
--------------------
- The HTMLElement interface represents any HTML element
    - Every HTMLElement inherits from an Element prototype
    - An Element has (inherits from) a Node prototype
    - Node has (inherits from) an EventTarget prototype

    - Reference (https://developer.mozilla.org/en-US/docs/Web/API/HTMLElement)


- HTML elements are a specific type of Nodes that
    - Brings their own properties & methods


- The HTMLInputElement interface
    - Provides special properties & methods for manipulating
        - The options, layouts & presentations of HTML element <input>

    - Every HTMLInputElement inherits from an HTMLElement prototype
    - An HTMLElement inherits from an Element prototype
    - An Element has a Node prototype
    - A Node has an EventTarget prototype

    - NOTE: It has access to ALL properties & methods of its "ancestors"
        - In addition to its own properties & methods
    
    - NOTE: An HTMLInputElement instance is "created" by simply
        - Selecting a HTML element <input> from the DOM
        - An HTMLInputElement object (instance) is now "created"
