Positioning The Tooltip
-----------------------
- Implement the logic on position the Tooltip of each "ProjectItem" object's "More Info" button   
    - Positioning the content near to their respective "More Info" button


- On the practice project (assets/scripts/app.js)
    - On the Sub-Class "Tooltip" Constructor method
        - Create a new parameter variable named "hostElementId"

        - Pass the parameter variable "hostElementId" as
            - An argument value to "super()"

        - NOTE: Identifies the corresponding Element node of a "ProjectItem"
            - Where the tooltip content resides c/o the attribute "data-extra-info"


    - On the Sub-Class "Tooltip" method "create()"
        - Prior to adding an Event listener to the local constant "tooltipElement"
            - Fetch the position of the inherited Class property "hostElement"
            - Chain the method ".getBoundingClientRect()"
            - Pass the statement as an argument value to "console.log()"

        - NOTE: Returns a "DOMRect" object that contains position-related information of
            - The selected host element stored in the local constant "hostElement" w/c
            - Includes height, width, x-coordinates, y-coordinates & more 

        
    - On the Class "ProjectItem" method "showMoreInfoHandler()"
        - Pass the Class property "id" as the 3rd argument value 
            - When instantiating the Class "Tooltip"

        - NOTE: Identifies the "ProjectItem" object whose "More Info" button is clicked
            - Renders the tooltip content just below the "More Info" button


    - On the Sub-Class "Tooltip" method "create()"
        - Comment out the recent "console.log()" of the Class property "hostElement"
            - Chained w/ the method ".getBoundingClientRect()"

        - Create a local constant named "hostElPosLeft"
            - Access the Class property "hostElement"
            - Chain the property ".offsetLeft"
            - Pass it as an assigned value 

            - NOTE: Refers to the distance of the host element's leftmost-point
                - From the start of the coordinate system (0,0)
    
        - Create a local constant named "hostElPosTop"
            - Access the Class property "hostElement"
            - Chain the property ".offsetTop"
            - Pass it as an assigned value 

            - NOTE: Refers to the distance of the host element's topmost-point
                - From the start of the coordinate system (0,0)

        - Create a local constant named "hostElHeight"
            - Access the Class property "hostElement"
            - Chain the property ".clientHeight"
            - Pass it as an assigned value

            - NOTE: Refers to the host element's height
                - Excluding borders and/or possible scroll bars 
                
        - Create a local constant named "parentElementScrolling"
            - Access the Class property "hostElement"
            - Chain the property ".parentElement"
            - Followed by the property ".scrollTop"
            - Pass it as an assigned value

            - NOTE: Holds a numeric value on how much of the host element's 
                - Parent (element) that has been scrolled down from its top-most point

        - Renders the tooltip content just below a "ProjectItem" object's "More Info" button

        - Next is to adjust the position of the tooltip content

        - Create a local constant named "x"
            - Assign the local constant "hostElPosLeft" as value
            - Then add the numeric value of 20 (i.e. 20 pixels)

        - Create a local constant named "y"
            - Assign the local constant "hostElPosTop" as value
            - Add the local constant "hostElHeight"
            - Subtract the local constant "parentElementScrolling" &
            - The numeric value of 10 (i.e. 10 pixels)

        - To apply the style changes on the tooltip content
            - Access the local constant "tooltipElement"
            - Chain the property ".style"
            - Followed by the property ".position"
            - Assign the value "absolute"

            - Access the local constant "tooltipElement"
            - Chain the property ".style"
            - Followed by the property ".left"
            - Assign the local constant "x" w/ the string "px" as value (e.g. x + "px")
            
            - NOTE: This mimics assigning values to CSS Selectors

            - Access the local constant "tooltipElement"
            - Chain the property ".style"
            - Followed by the property ".top"
            - Assign the local constant "y" w/ the string "px" as value (e.g. y + "px")

            - NOTE: This mimics assigning values to CSS Selector properties
        
    
    - Save the changes
        - Load / reload the browser

        - Clicking a "ProjectItem" object's "More Info" button
            - Renders the tooltip content at a considerable distance from the button

        - NOTE: The tooltip content is currently NOT sticky
            - Scrolling down / up on each project group (I.e. Active or Finished Projects)
            - Leaves the corresponding tooltip content in-place
            - Instead of being "stuck" w/ its "More Info" button
