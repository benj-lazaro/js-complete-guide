The "Event" Object
-------------------
- The "Event" object
    - Generated automatically by the browser
    - Then passed to an Event handler
    - NOTE: This is the core idea behind event-driven code
    
    - Whatever triggered an event, it (typically) provide
        - Data (metadata) to descibe it
        - Methods to control how it behaves


- The "Event" object property "target"
    - Describes the Element node that triggered an event (e.g. "click")
    - Identifies & grants direct access to the said Element node


- The "Event" object property "target.disabled"
    - Disables the Element node triggered by an event
