Closures
--------
- Every JavaScript function is a Closure
    - A function "closes over" the variable(s) defined w/in its own environment
        - Registers local variable(s)
        - Memorize (locks in) assigned values

    - Each function has its own "lexical environment" (i.e. scope)
        - An inner function (w/in a Factory function) has access to
            - ALL parameters & variables defined on the outer function
            - ALL variables defined globally

        - An outer function has NO access to
            - Constants, parameters & varibales defined w/in an inner function

        - NOTE: Every call to the outer function
            - Locks in the assigned values (parameters and/or variables)
            - A new (detached) inner function is created & then executed

        - A global variable is accessed by the outer and/or inner function
            - It does NOT get "locked in" by either functions
            - Any changes in the assigned value
            - Either functions (inner or outer) picks up the latestt assigned value
