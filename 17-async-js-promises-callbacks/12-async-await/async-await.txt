Async / Await
-------------
- Both "async" & "await" serves as an alternative to Promises
    - Utilizes normal Promise
    - Omits the need to use of methods ".then()" & ".catch()"
    

- Place the keyword "async" in front of a function name (or function declarations)
    - Enables a function to return a "Promise" object
        - This also effectively wraps the entire function in a "Promise" object

    - NOTE: On function expressions, the keyword "async" is placed
        - On the right-hand side of the "=" operator


- Inside of the async function
    - Place the keyword "await" in front of any inner "Promise" object
        - It waits for the inner "Promise" object to be resolved or rejected

        - Returns the corresponding data or error
            - NOTE: Save the returned value to a local constant

        - Executes the succeeding line of code


- NOTE: They "async" & "await" keywords does NOT change the way JavaScript works
    - It still utilizes the normal Promise
    - Although performs the method ".then()" behind-the-scenes
